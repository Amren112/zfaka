## ZFAKA安装教程（Docker版）：

Docker版优势

**优点**

- 30s即可成功搭建一个zfaka（熟悉docker的人）
- 新手从0开始，也可以快速搭建，全自动部署
- 不用手动去配置yaf、扩展插件、伪静态等等
- 采用前后端数据分离、更安全
## 一键脚本
```
bash <(curl -L -s https://raw.githubusercontent.com/Baiyuetribe/zfaka/master/zfaka.sh)
```
